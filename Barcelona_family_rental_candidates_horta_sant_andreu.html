筛选目标

2–3房、长期住宅合同、可全家登记、靠近学校和交通。

重点区域

Horta / Llobregós / Maragall / Can Baró / El Guinardó / Sant Andreu de Palomar / Navas / La Sagrera / Congrés i els Indians。

核心风险

很多低价房源可能是 temporada（短租/临时租赁），不适合入学主地址。

先看结论

最优先：Horta 核心、Llobregós / Passeig Maragall、Sant Andreu de Palomar / Fabra i Puig。
最谨慎：El Carmel 深处、Bon Pastor / Trinitat Vella 部分街段、所有标注 temporada 的房源。
签约红线：必须确认合同为 vivienda habitual，且允许全家 empadronamiento。

联系中介/房东必须问

  • ¿Es contrato de vivienda habitual?（是长期自住房合同吗？）
  • ¿Se puede empadronar toda la familia?（全家可以做住址登记吗？）
  • ¿La duración inicial es de 12 meses o más?（初始合同是否 12 个月或以上？）
  • ¿No es contrato de temporada?（不是短租/临时合同，对吗？）
  • ¿Qué gastos están incluidos?（包含哪些费用？）
  • ¿Hay ascensor?（有电梯吗？）
  • ¿Se aceptan niños/familias?（接受家庭和孩子入住吗？）
  • ¿Cuándo se puede visitar?（什么时候可以看房？）

给你的实际筛选顺序

  1. 先筛合同：直接排除 temporada / vacacional / estancia temporal。
  2. 再筛登记：问清能否全家 empadronamiento。
  3. 再筛学校：Horta-Guinardó 对 Piaget / Guinardó / FEDAC-Horta；Sant Andreu 对 Arrels / FEP L'Esperança。
  4. 最后看通勤：实地走一次早高峰路线，尤其是坡度、噪音和夜间安全。
