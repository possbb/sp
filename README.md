# Spain Climate Site

This folder is a static website package. Upload all files and folders in this directory to your cloud server or static hosting service.

## Entry point
- `index.html`

## Main pages
- `Barcelona_climate_trend_analysis_1990_2026.html`
- `Barcelona_extreme_weather_timeline.html`
- `Barcelona_extreme_weather_map.html`
- `Beijing_vs_Barcelona_climate_comparison_1990_2030.html`
- `charts.html`
- `Barcelona_Beijing_haze_comparison_2021_2025.html`
- `Barcelona_primary_school_district_guide.html`
- `Barcelona_primary_school_map.html`

## Data files
- `Barcelona_climate_monthly_1990_2026.xlsx`
- `Beijing_climate_monthly_1990_2026.xlsx`
- `Barcelona_Beijing_air_quality_2021_2025.xlsx`
- `Barcelona_primary_schools_by_district_2025.xlsx`

## Deployment notes
- Keep all files and folders in the same directory.
- Use `index.html` as the default landing page.
- The site is static and does not require a backend.
- Configure the server to serve `.xlsx` as downloadable files if needed.
